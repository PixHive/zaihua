# 在花(@ZAIHUA) - 个人主页

## 项目简介

这是一个现代化的个人主页项目。

## 项目结构

```
/
├── index.html          # 主页

├── 404.html           # 404错误页面
├── css/
│   ├── style.css      # 主样式文件

├── js/
│   └── common.js      # 公共JavaScript功能
└── imgs/
    ├── avatar.jpg     # 头像
    ├── wallpaper.png  # 背景图
    └── favicon/       # 网站图标
```
